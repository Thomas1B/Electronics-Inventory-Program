# Electronics Inventory Program

This program is used to keep track of electronics parts, order sheets, projects, and other things.


The "gui_app" and "data_handling" modules can be found in the "program_modules" folder of this program. The "data_handling" module uses pandas (https://pandas.pydata.org/) to handle the data, while the "gui_app" module runs the GUI using PyQt5. The GUI was designed with PyQt Designer, available for download at: https://build-system.fman.io/qt-designer-download.
